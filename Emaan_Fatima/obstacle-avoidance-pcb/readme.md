
# Obstacle Avoidance Robot

## Project Overview

This project contains the Tinkercad simulation files for an Obstacle Avoidance Robot, including Arduino code, circuit diagram, and demo recording of the working simulation.

## Files Included

- `obstacle_avoidance.ino` - Arduino Source Code
- `circuit_diagram.jpg` - Circuit Layout Screenshot
- `demo_video.mp4` - Working Demo Recording

## Tools Used

- Tinkercad Circuits
