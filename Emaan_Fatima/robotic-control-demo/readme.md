# Robotic Control

This project contains the Tinkercad simulation files for a basic Robotic Control system, including Arduino code, circuit diagram, and demo recording of the working simulation.

## Files Included

- `robotic_control.ino` - Arduino Source Code
- `circuit_diagram.jpg` - Circuit Layout Screenshot
- `demo_video.mp4` - Working Demo Recording

## How It Works

The Arduino Uno controls two DC motors through an L298N motor driver module, powered by a 9V battery. The Arduino sends control signals to the motor driver, which regulates the direction and speed of both motors, allowing the robot to move forward, backward, and turn.
